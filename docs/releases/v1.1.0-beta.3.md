# v1.1.0-beta.3 - Claude model selection beta / Claude 模型选择 beta

This beta release adds selectable Claude Code model configuration to the one-click setup page, so users can choose the routed models they want instead of being limited to fixed provider presets.
本次 beta 版本为一键配置页新增 Claude Code 模型选择能力，用户可以按需选择要写入的路由模型，不再只能使用固定厂商预设。

##  Features / 新特性

- **Selectable Claude Models**: Added a Claude Code model picker in one-click setup with a hard limit of 4 selected models. @mibgb65-cloud
  - 在一键配置中新增 Claude Code 模型选择器，并强制限制最多选择 4 个模型。
- **Mixed Provider Slots**: Selected DeepSeek, MiMo, Kimi, and GLM models can be written into Claude Code model slots through OmniProxy's Anthropic router. @mibgb65-cloud
  - 支持将选择的 DeepSeek、MiMo、Kimi、GLM 模型写入 Claude Code 模型槽位，并通过 OmniProxy Anthropic Router 路由。
- **Desktop and HTTP API Support**: Added Wails and local HTTP API support for custom Claude model configuration. @mibgb65-cloud
  - 新增桌面 Wails 调用和本地 HTTP API，支持自定义 Claude 模型配置。

##  Bug Fixes / 问题修复

- **No Default Selection**: The new Claude model picker now starts empty, so users must explicitly choose the models they want to configure. @mibgb65-cloud
  - 新增的 Claude 模型选择器默认不再预选模型，用户需要明确选择要配置的模型。
- **Backend Validation**: Model selection is validated on the backend to reject empty selections, unsupported model IDs, and more than 4 unique models. @mibgb65-cloud
  - 后端会校验空选择、不支持的模型 ID，以及超过 4 个去重模型的请求。

## ⚠️ Breaking Changes / 不兼容更新

- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。

##  Refactor & Performance / 重构与优化

- **Claude Settings Writer**: Added reusable Claude model slot writing logic and tests for selected model combinations. @mibgb65-cloud
  - 新增可复用的 Claude 模型槽位写入逻辑，并补充选择组合的测试覆盖。
- **Generated Bindings**: Updated Wails generated bindings for the new Claude model configuration request and response fields. @mibgb65-cloud
  - 更新 Wails 生成绑定，支持新的 Claude 模型配置请求和响应字段。
