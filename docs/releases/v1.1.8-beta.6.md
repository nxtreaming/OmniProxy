# v1.1.8-beta.6 - unified update ready prompt / 统一更新准备完成提示

This beta continues the `v1.1.8` validation line with a cleaner update-ready interaction. After an update installer is downloaded and verified, OmniProxy now keeps the restart action inside the titlebar update popover instead of opening a separate centered confirmation dialog.

本次 beta 继续验证 `v1.1.8` 版本线，优化更新安装包准备完成后的提示体验。更新安装包下载并校验完成后，OmniProxy 现在会把重启安装操作保留在标题栏更新悬浮卡片中，不再额外弹出居中的确认窗口。

## Improvements / 优化

- **Unified update prompt**: The update-ready state now appears inside the existing titlebar update popover, keeping update status and actions in one place.
  - **统一更新提示**：更新准备完成状态现在直接显示在标题栏更新悬浮卡片中，让更新状态与操作入口保持一致。
- **No duplicate confirmation dialog**: Clicking `重启安装` from the popover starts the install flow directly, without another centered confirmation dialog.
  - **取消重复确认弹窗**：从悬浮卡片点击 `重启安装` 会直接进入安装流程，不再出现第二个居中确认弹窗。
- **Clear ready-state copy**: The popover now uses `Beta 已准备好 / 新版本已准备好` wording and explains that OmniProxy will close, launch the installer, and reopen after installation.
  - **准备完成文案更明确**：悬浮卡片现在使用 `Beta 已准备好 / 新版本已准备好` 文案，并说明将关闭 OmniProxy、启动安装器、安装完成后重新打开应用。

## Compatibility / 兼容性

- **Attention**: This release changes only frontend update UI behavior. Update download, checksum verification, installer launch, and version comparison behavior are unchanged.
  - **必读**：此版本仅调整前端更新提示交互。更新下载、SHA256 校验、安装器启动和版本比较行为均不变。
