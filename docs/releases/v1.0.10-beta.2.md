# v1.0.10-beta.2 - Mock billing beta / 模拟账单 beta

This beta release focuses on the mock billing experience in OmniProxy. It adds richer bill image templates, bill-content language switching, and a preview-before-save flow for validating generated PNG output.
本次 beta 版本聚焦 OmniProxy 的模拟账单体验，加入更多账单图片模板、账单内容中英文切换，以及保存前预览生成 PNG 的流程。

## Beta Notes / Beta 说明
- **Prerelease**: This build is intended for UI and billing-image validation before the stable `v1.0.10` release.
  - **预发布**: 此版本用于正式 `v1.0.10` 前验证界面和账单图片生成效果。
- **Mock Billing**: Generated bill images are only for local review and sharing, not official reconciliation.
  - **模拟账单**: 生成的账单图片只用于本地回顾和分享，不作为正式对账凭证。

## Changes / 更新内容
- **Bill Templates**: Add multiple mock bill styles including poster, neon, receipt, clean card, meter, and longform data layouts.
  - **账单模板**: 新增费用海报、午夜霓虹、复古小票、清爽卡片、能量仪表和数据长图等多种模拟账单风格。
- **Bill Content Language**: Add Chinese and English switching for the generated bill content while keeping the app page UI in Chinese.
  - **账单内容语言**: 生成账单内容支持中文和英文切换，同时应用页面本身保持中文界面。
- **Preview Before Save**: Show the generated bill image in a modal preview before saving it as PNG.
  - **保存前预览**: 保存 PNG 前先展示生成后的账单图片，方便确认模板、日期和语言。
- **Layout Refinement**: Reorganize the billing header so template title and language selector stay on the first row, with date and actions on the second row.
  - **布局调整**: 调整账单顶部区域，第一行放「模拟账单」和账单内容语言，第二行左侧放日期，右侧放刷新和预览。

## Bug Fixes / 问题修复
- **Preview Stability**: Avoid page blanking after bill preview or image export by keeping the preview inside the billing view and using a safer save flow.
  - **预览稳定性**: 调整预览弹窗和保存流程，避免预览或导出账单图片后其它页面内容变空。

## Breaking Changes / 不兼容更新
- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。
