# v1.0.8 - DeepSeek balance currency fix / DeepSeek 多币种余额修复

This patch release fixes DeepSeek balance display for accounts that return both CNY and USD balance entries.
本次补丁版本修复 DeepSeek 国际版账号同时返回 CNY 与 USD 余额时的展示和可用性判断问题。

## Bug Fixes / 问题修复
- **DeepSeek Balance**: Parse all `balance_infos` entries and choose a positive balance deterministically, instead of relying on the first returned currency. Fixes #2. @mibgb65-cloud
  - 解析 DeepSeek 返回的所有 `balance_infos` 项，优先选择有余额的币种，避免上游返回顺序变化时误显示为 `0`。修复 #2。
- **DeepSeek Balance Fallback**: When `total_balance` is missing, fall back to `granted_balance + topped_up_balance`.
  - 当 `total_balance` 缺失时，使用 `granted_balance + topped_up_balance` 计算余额。

## Maintenance / 维护
- Removed the one-off workflow that commented on and closed Issue #1.
  - 移除一次性的 Issue #1 自动评论和关闭 workflow。

## Breaking Changes / 不兼容更新
- **Attention**: No breaking changes are expected in this release.
  - **必读**: 本版本预计没有不兼容更新。
