# v1.2.1-beta.1 - Codex Team quota label fix / Codex Team 额度标签修复

This beta starts the `v1.2.1` validation line with a focused desktop UI fix for Codex quota display. Codex Team accounts now show their primary quota window as a monthly quota instead of using the 5-hour quota label intended for Plus / Pro style Codex windows.

本次 beta 开启 `v1.2.1` 验证版本线，集中修复 Codex 额度展示。Codex Team 账号的主额度窗口现在显示为月度额度，不再误用面向 Plus / Pro 类 Codex 窗口的 5 小时额度标签。

## Fixes / 修复

- **Codex Team monthly quota label**: Codex `team` plan accounts now show `本月额度` for the primary quota window.
  - **Codex Team 月度额度标签**：Codex `team` 套餐账号的主额度窗口现在显示为 `本月额度`。
- **Codex Plus / Pro behavior preserved**: Non-Team Codex subscription windows continue to show the existing `5h额度` and `1 周额度` labels.
  - **保留 Codex Plus / Pro 行为**：非 Team 的 Codex 订阅窗口继续沿用现有 `5h额度` 与 `1 周额度` 标签。
- **Regression coverage**: Added a frontend utility test to lock the Codex Team label behavior while keeping Plus unchanged.
  - **回归覆盖**：新增前端工具函数测试，锁定 Codex Team 标签行为，并确认 Plus 展示不变。

## Compatibility / 兼容性

- **Attention**: This release changes frontend display text only. Proxy routing, scheduling, credential storage, quota refresh parsing, and local data formats are unchanged.
  - **必读**：本版本仅调整前端展示文案。代理路由、账号调度、凭据存储、额度刷新解析和本地数据格式均未改变。
