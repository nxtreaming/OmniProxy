# v1.0.9-beta.1 - Billing reports and safer in-app updates / 账单报告与更安全的应用内更新

This beta release adds local billing estimates, improves update downloads with SHA256 verification, and expands Xiaomi MiMo Token Plan routing for overseas SGP accounts.
本次 beta 版本新增本地费用账单估算，支持应用内下载并校验更新安装包，同时补充 Xiaomi MiMo Token Plan 海外 SGP 账号路由。

## Beta Notes / Beta 说明
- **Prerelease**: This build is intended for validation before the stable `v1.0.9` release.
  - **预发布**: 此版本用于正式 `v1.0.9` 前的功能验证。
- **Update Flow**: Beta users should be able to detect the later stable `v1.0.9` release as an available update.
  - **更新流程**: beta 用户后续应能检测到正式 `v1.0.9` 可用更新。

## Features / 新特性
- **Billing Reports**: Add a billing page that estimates daily usage cost from local request history and exports an invoice-style PNG report.
  - 新增「费用账单」页面，可根据本地请求历史按模型估算每日费用，并导出发票风格的 PNG 报告图。
- **In-app Updates**: Download release installers inside OmniProxy, track progress, verify SHA256 checksums, and launch the verified installer.
  - 支持在 OmniProxy 内下载发布安装包、显示进度、校验 SHA256，并启动已校验的安装器。
- **Xiaomi MiMo SGP**: Support China and overseas SGP regions for MiMo Token Plan accounts, including separate OpenAI-compatible and Anthropic base URLs.
  - MiMo Token Plan 账号新增中国区与海外 SGP 区域选择，并分别支持 OpenAI 兼容和 Anthropic Base URL。

## Maintenance / 维护
- **Agent Guide**: Add `AGENTS.md` with project structure, commands, validation guidance, and credential handling rules for Codex-style agents.
  - 新增 `AGENTS.md`，说明项目结构、常用命令、验证方式和凭据处理规则，方便 Codex 类编程代理协作。

## Breaking Changes / 不兼容更新
- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。
