# v1.1.6-beta.1 - dashboard card alignment polish / 仪表盘卡片对齐优化

This beta starts the `v1.1.6` line with focused dashboard layout fixes. It keeps the Gemini-style desktop refresh from `v1.1.5` and refines the account-usage and quota-overview cards reported after the stable release.

本次 beta 开始 `v1.1.6` 版本线，集中修复仪表盘布局细节。在保留 `v1.1.5` Gemini 风格桌面重构的基础上，优化稳定版发布后反馈的账号占用和额度概览卡片显示问题。

## Improvements / 优化

- **Tool usage card density**: When only one coding tool is active or recently used, the "coding tool account usage" card now keeps the row compact instead of stretching content across the whole card height.
  - **工具占用卡片密度**：当只有一个编程工具正在使用或最近使用时，「编程工具账号占用」卡片会保持紧凑行高，不再把内容拉散到整张卡片上下两端。
- **Quota overview alignment**: The subscription quota and API/balance columns now use matched inner containers, making the left and right sides align visually even when one side has fewer rows.
  - **额度概览对齐**：订阅额度与 API / 余额状态改为一致的内部容器高度，即使一侧数据较少，左右卡片也会保持视觉对齐。

## Bug Fixes / 问题修复

- **Dashboard single-row layout**: Fixed the single tool-usage row layout that could appear overly sparse in the dashboard.
  - **仪表盘单行布局**：修复工具占用只有一条记录时显示过于分散的问题。
- **Quota empty-state balance**: Fixed uneven quota overview height caused by different row counts or empty states.
  - **额度空状态平衡**：修复额度概览因左右数据条数不同或空状态导致高度不一致的问题。

## Compatibility / 兼容性

- **Attention**: No backend API, proxy route, credential format, or local data format breaking changes are expected in this beta release.
  - **必读**：本 beta 预计没有后端 API、代理路由、凭据格式或本地数据格式不兼容更新。
