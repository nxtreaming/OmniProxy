# v1.1.7-beta.2 - tray menu and quota feedback polish / 托盘菜单与额度反馈优化

This beta continues the `v1.1.7` line with focused UI polish for quota workflows and the Windows tray menu. It replaces the native tray popup menu with a custom dark rounded popup so the menu no longer shows the Windows white border and sharp corners.

本次 beta 继续完善 `v1.1.7` 版本线，集中优化额度流程和 Windows 托盘菜单。托盘菜单已从系统原生弹出菜单改为自绘暗色圆角弹层，避免出现 Windows 默认白边和直角外框。

## Improvements / 优化

- **Custom tray menu**: The Windows tray menu now uses an OmniProxy-styled dark rounded popup with a quiet status header, subtle separators, and soft hover states.
  - **自绘托盘菜单**：Windows 托盘菜单现在使用符合 OmniProxy 风格的暗色圆角弹层，包含克制的状态头部、弱分割线和柔和 hover 状态。
- **Quota refresh progress**: The quota page shows a top progress toast during "refresh all" with animated multicolor border feedback, while regular success and error messages stay at the bottom.
  - **额度刷新进度**：额度页点击“全部刷新”时会在页面上方显示带多彩边框动画的进度提示，普通成功和错误提示仍保留在底部。
- **Current quota card polish**: The active quota card style has been toned down to better match the app's Gemini-inspired dark UI.
  - **当前额度卡片优化**：当前使用的额度卡片样式已收敛为更贴合 Gemini 暗色风格的视觉表现。

## Bug Fixes / 问题修复

- **Tray menu frame**: Removed the native Windows popup menu frame that caused a bright white border and rigid 90-degree corners in dark mode.
  - **托盘菜单外框**：移除原生 Windows 弹出菜单外框，修复暗色模式下出现亮白边和硬直角的问题。

## Compatibility / 兼容性

- **Attention**: No proxy route, credential format, local data format, or Wails binding changes are expected in this beta release.
  - **必读**：本 beta 预计没有代理路由、凭据格式、本地数据格式或 Wails 绑定不兼容更新。
