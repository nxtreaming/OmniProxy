# v1.2.0-beta.2 - macOS preview release workflow hardening / macOS 预览发布流程加固

This beta rebuilds the `v1.2.0` macOS preview release after hardening the GitHub Actions release workflow against action download failures and Node.js 20 deprecation warnings.

本次 beta 在加固 GitHub Actions 发布流程后重新构建 `v1.2.0` macOS 预览发布，规避 action 下载失败，并处理 Node.js 20 action 弃用警告。

## Changes / 变更

- **Release workflow hardening**: Enabled Node.js 24 runtime opt-in for JavaScript actions, removed the release dependency on `softprops/action-gh-release`, and replaced `actions/setup-go` with direct Go toolchain preparation where needed.
  - **发布流程加固**：为 JavaScript actions 启用 Node.js 24 运行时 opt-in，移除对 `softprops/action-gh-release` 的发布依赖，并在需要时用直接准备 Go toolchain 的方式替代 `actions/setup-go`。
- **Release scope unchanged**: The application functionality remains the same as `v1.2.0-beta.1`, including Windows installer output and unsigned macOS universal DMG output.
  - **发布范围不变**：应用功能与 `v1.2.0-beta.1` 保持一致，包括 Windows 安装包和未正式签名的 macOS universal DMG。

## Validation / 验证

- GitHub Actions release workflow with Node.js 24 runtime opt-in.
- Windows installer build.
- macOS backend tests, frontend tests, universal app build, and unsigned DMG packaging.
