# v1.2.0-beta.2 - GitHub Actions Node 24 runtime update / GitHub Actions Node 24 运行时更新

This beta rebuilds the `v1.2.0` macOS preview release with GitHub Actions upgraded from Node.js 20-based actions to Node.js 24-compatible action versions.

本次 beta 使用已升级到 Node.js 24 兼容版本的 GitHub Actions 重新构建 `v1.2.0` macOS 预览发布，避免 Node.js 20 action 弃用警告。

## Changes / 变更

- **CI action runtime update**: Upgraded `actions/checkout`, `actions/setup-go`, `actions/setup-node`, `actions/upload-artifact`, and `softprops/action-gh-release` to Node.js 24-compatible major versions.
  - **CI action 运行时更新**：升级 `actions/checkout`、`actions/setup-go`、`actions/setup-node`、`actions/upload-artifact` 和 `softprops/action-gh-release` 到兼容 Node.js 24 的主版本。
- **Release scope unchanged**: The application functionality remains the same as `v1.2.0-beta.1`, including Windows installer output and unsigned macOS universal DMG output.
  - **发布范围不变**：应用功能与 `v1.2.0-beta.1` 保持一致，包括 Windows 安装包和未正式签名的 macOS universal DMG。

## Validation / 验证

- GitHub Actions release workflow with Node.js 24-compatible actions.
- Windows installer build.
- macOS backend tests, frontend tests, universal app build, and unsigned DMG packaging.
