# v1.1.7-beta.5 - Settings layout and update badge / 设置页布局与更新提示

This beta continues the `v1.1.7` validation line with fixes for the global settings page and update discovery UI.

本次 beta 继续完善 `v1.1.7` 验证版本线，重点修复全局设置页布局和更新发现入口。

## Improvements / 优化

- **Global settings layout**: Reworked the global settings page into explicit left and right columns, so a taller "放心刷" card no longer pushes unrelated sections down and leaves a large blank area.
  - **全局设置页布局**：将全局设置页改为明确的左右分栏，避免「放心刷」卡片较高时撑大整行并把其它模块挤到下方。
- **About page update action**: Aligned the "检查更新" action to the top of the version area instead of sitting low beside the version number.
  - **关于页更新按钮**：将「检查更新」操作对齐到版本区域顶部，避免按钮贴在大号版本号下缘。
- **Titlebar update hint**: Added a compact "新版本" pill in the titlebar when the existing update check reports that a newer release is available. Clicking it opens the About page.
  - **标题栏更新提示**：复用现有更新检查结果，在发现新版本时于标题栏显示「新版本」提示，点击可进入「关于应用」。
