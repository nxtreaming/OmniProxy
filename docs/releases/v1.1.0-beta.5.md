# v1.1.0-beta.5 - Free quota mapping and README polish / Free 额度映射与 README 优化

This beta release fixes a Free-plan quota display regression left in `v1.1.0-beta.4` and refreshes the Chinese README into a more complete open-source project landing page.
本次 beta 版本修复 `v1.1.0-beta.4` 遗留的 Free 账号额度显示回归，并将中文 README 优化为更完整的开源项目首页。

##  Features / 新特性

- **README Redesign**: Reworked the Chinese README with a clearer project pitch, badges, architecture diagram, capability matrix, local route table, provider support matrix, release channel notes, and security model. @mibgb65-cloud
  - 重写中文 README，补充项目定位、徽章、架构图、能力矩阵、本地入口、厂商支持矩阵、发布通道和安全模型。

##  Bug Fixes / 问题修复

- **Codex Free Window Mapping**: Free-plan Codex usage that arrives in `primary_window` is now mapped to the weekly quota fields instead of being treated as a 5h quota. @mibgb65-cloud
  - 当 Codex Free 账号的用量从 `primary_window` 返回时，现在会映射到周额度字段，不再当作 5h 额度。
- **Quota Card Visibility**: Empty quota windows with zero values are no longer displayed, preventing Free accounts from showing an extra `1 周额度 0%` card. @mibgb65-cloud
  - 空的 0 值额度窗口不再展示，避免 Free 账号多显示一个 `1 周额度 0%` 卡片。
- **Free Quota Label**: Existing Free-plan data stored in the primary quota fields is displayed as `1 周额度`, so previously refreshed accounts no longer show `5h额度`. @mibgb65-cloud
  - 已落在 primary 字段里的 Free 账号旧数据会显示为 `1 周额度`，不再显示 `5h额度`。
