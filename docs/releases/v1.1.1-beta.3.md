# v1.1.1-beta.3 - sub2api image intent guard / sub2api 图片意图防护

This beta patch hardens the local sub2api adapter for newer Codex Responses payloads. It removes image-generation tool metadata more completely before forwarding ordinary text requests to sub2api, avoiding false group permission rejections.
本次 beta 补丁加强本地 sub2api 适配器对新版 Codex Responses 请求体的兼容处理。在转发普通文本请求前，会更完整地移除图片生成工具元数据，避免被 sub2api 分组图片权限误判拦截。

## Features / 新特性
- None / 无。

## Bug Fixes / 问题修复

- **sub2api Codex Responses**: Recursively strips `image_generation` tools and `tool_choice` fields from sub2api OpenAI Responses requests.
  - **sub2api Codex Responses**: 对 sub2api OpenAI Responses 请求递归移除 `image_generation` 工具和指向它的 `tool_choice` 字段。
- **sub2api Include Filtering**: Removes image-generation include entries such as partial image output selectors while preserving normal text and non-image tools.
  - **sub2api Include 过滤**: 移除图片生成相关 include 项，例如 partial image 输出选择项，同时保留普通文本和非图片工具。

  - **必读**: 本 beta 版本预计没有后端 API 或本地数据格式不兼容更新。
