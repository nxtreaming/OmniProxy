# v1.1.1-beta.4 - token management import tools / 账号管理导入工具

This beta adds account-management helpers for local credential pools. It introduces batch API Key import and manual Codex OAuth token refresh from the OpenAI account table.
本次 beta 增加账号池管理辅助功能：支持批量导入 API Key，并可在 OpenAI 账号列表中手动刷新 Codex OAuth 令牌。

## Features / 新特性

- **Batch API Key Import**: Adds a batch import dialog in account management. Each non-empty line imports the first field as an API Key and ignores inline comments after `#`.
  - **批量 API Key 导入**: 账号管理新增批量导入弹窗。每个非空行导入第一个字段作为 API Key，并忽略 `#` 后的行内备注。
- **Automatic Names**: Imported API Key accounts are named from the first 8 characters of the key, with suffixes added for duplicate names.
  - **自动命名**: 导入的 API Key 账号默认使用 Key 前 8 位命名，名称冲突时自动追加后缀。
- **sub2api Batch Import**: sub2api batch import accepts a per-batch Base URL so keys from different upstreams can coexist.
  - **sub2api 批量导入**: sub2api 批量导入支持按批次填写 Base URL，方便不同上游的 Key 共存。
- **Codex Token Refresh Button**: OpenAI Codex auth.json accounts now show a manual refresh-token button.
  - **Codex 令牌刷新按钮**: OpenAI 的 Codex auth.json 账号新增手动刷新令牌按钮。
