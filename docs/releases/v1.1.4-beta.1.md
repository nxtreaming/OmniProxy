# v1.1.4-beta.1 - first-run onboarding and client endpoint reference / 首次引导与客户端接口速查

This beta starts the `v1.1.4` series with clearer first-use guidance, broader third-party client connection references, and Zo Claude model support for both Claude Code and Claude Desktop routing.

本次 beta 开始 `v1.1.4` 系列，重点优化首次使用引导、第三方客户端接口速查，并让 Claude Code 与 Claude Desktop 都可以使用 Zo Claude 模型路由。

## Features / 新特性

- **First-Run Guide**: Added a one-time global onboarding dialog that walks new users through adding accounts, starting the local proxy, and opening one-click client setup.
  - **首次使用引导**：新增首次打开时自动显示一次的全局引导框，按步骤提示添加账号、启动本地代理和进入一键配置。
- **Client Endpoint Reference**: Added a full third-party client endpoint reference in the help page, including OpenAI-compatible, Anthropic-compatible, vendor-specific, and coding-tool routes with copy actions.
  - **客户端接口速查**：在使用说明中新增完整第三方客户端接口清单，覆盖 OpenAI 兼容、Anthropic 兼容、厂商直连和编程工具路由，并支持复制配置值。
- **Zo Claude Routing**: Added Zo Claude Opus 4.7 and Sonnet 4.6 options to Claude Code and Claude Desktop model configuration.
  - **Zo Claude 路由**：Claude Code 与 Claude Desktop 模型配置新增 Zo Claude Opus 4.7 和 Sonnet 4.6 选项。
- **README Release Channel Notes**: Updated README release-channel guidance for the `v1.1.4-beta.1` beta track.
  - **README 发布通道说明**：更新 README 中的发布通道说明，标明 `v1.1.4-beta.1` beta 版本线。

## Bug Fixes / 问题修复

- **Claude Desktop Zo Gateway**: Fixed Claude Desktop gateway requests for Zo Claude models so they are routed through the Zo provider instead of the default Anthropic provider.
  - **Claude Desktop Zo 网关**：修复 Claude Desktop 使用 Zo Claude 模型时会落到默认 Anthropic provider 的问题，确保请求进入 Zo provider。
- **About Page Update Button**: Fixed the check-update button text color in disabled/development states.
  - **关于页检查更新按钮**：修复检查更新按钮在禁用或开发构建状态下文字颜色不清晰的问题。
- **Help Page Layout**: Reduced nested panels in the usage guide so the header and detailed sections render as separate, lighter boxes.
  - **使用说明布局**：减少使用说明页的方框嵌套，让标题说明和下面的详细内容分别以更轻量的独立方框展示。

## Breaking Changes / 不兼容更新

- **Attention**: No backend API or local data format breaking changes are expected in this beta release.
  - **必读**：本 beta 版本预计没有后端 API 或本地数据格式不兼容更新。
