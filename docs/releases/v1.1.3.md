# v1.1.3 - Zo Computer gateway and billing polish / Zo Computer 网关与账单优化

This release adds Zo Computer gateway support, expands one-click coding client configuration, improves API-key balance visibility, and polishes the billing report UI.

本次正式版新增 Zo Computer 网关，扩展编码客户端一键配置，优化 API Key 余额展示，并完善费用账单界面。

## Features / 新特性

- **Zo Computer Provider**: Added a Go-native Zo Computer gateway with OpenAI Chat Completions, OpenAI Responses, Anthropic Messages, and compatible model-list endpoints.
  - **Zo Computer Provider**：新增 Go 原生 Zo Computer 网关，支持 OpenAI Chat Completions、OpenAI Responses、Anthropic Messages 和模型列表兼容接口。
- **Codex and Claude Code Zo Setup**: Added one-click Zo Computer configuration for Codex and Claude Code, including Opus 4.7 and Sonnet 4.6 model slots.
  - **Codex 和 Claude Code Zo 配置**：新增 Zo Computer 一键配置，并写入 Opus 4.7 与 Sonnet 4.6 模型档位。
- **OpenCode and Pi Zo Models**: Added Zo Computer presets for GPT-5.5, GPT-5.4, GLM 5, Gemini 3.1 Pro, MiniMax 2.7, DeepSeek V4 Pro, and Claude models.
  - **OpenCode 和 Pi Zo 模型**：新增 GPT-5.5、GPT-5.4、GLM 5、Gemini 3.1 Pro、MiniMax 2.7、DeepSeek V4 Pro 和 Claude 模型预设。
- **API Balance Totals**: Provider quota and account pages now show API Key balances grouped by currency.
  - **API 余额汇总**：厂商额度页和账号管理页支持按币种汇总 API Key 余额。
- **Client Setup Shortcuts**: Added Claude Desktop and DeepSeek-TUI gateway configuration support.
  - **客户端配置快捷入口**：新增 Claude Desktop 和 DeepSeek-TUI 网关配置支持。

## Fixes / 修复

- **Codex Responses Compatibility**: Fixed Codex requests that send zstd-compressed request bodies to the local Zo Responses endpoint.
  - **Codex Responses 兼容性**：修复 Codex 向本地 Zo Responses 端点发送 zstd 压缩请求体时的 JSON 解析错误。
- **Responses Streaming Shape**: Improved synthetic OpenAI Responses streaming events so Codex can track active output items correctly.
  - **Responses 流式事件**：优化合成的 OpenAI Responses 流式事件，确保 Codex 能正确跟踪输出项。
- **Zo Model Matching**: Improved Zo model alias matching for vendor-prefixed and label-based model names.
  - **Zo 模型匹配**：优化 Zo 模型别名匹配，兼容带厂商前缀和显示标签的模型名称。
- **Billing Report UI**: Filled the billing detail sidebar with cost insights, model share bars, and ignored-model summaries; fixed poster preview colors in dark mode.
  - **费用账单界面**：右侧明细区新增账单洞察、模型占比和未纳入模型摘要，并修复暗色模式下海报预览发灰的问题。
- **Dashboard Recent Logs**: Fixed compact recent-log rows that could squeeze model, client, message, and timestamp text together.
  - **仪表盘最近日志**：修复紧凑日志行中文字和标签拥挤的问题。
