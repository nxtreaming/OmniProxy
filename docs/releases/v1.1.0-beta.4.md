# v1.1.0-beta.4 - Codex Free quota display beta / Codex Free 额度显示 beta

This beta release fixes Codex Free account quota display and refresh behavior when the upstream usage endpoint only returns the weekly quota window.
本次 beta 版本修复 Codex Free 账号在上游仅返回周额度窗口时的额度展示和刷新状态。

##  Features / 新特性

- **README Refresh**: Updated the README with recent beta changes, including Claude model selection, Pi single-entry routing, TokenRouter, OpenRouter desktop chat, and Codex Free quota display. @mibgb65-cloud
  - 更新 README，补充近期 beta 变化，包括 Claude 模型选择、Pi 单入口路由、TokenRouter、OpenRouter 桌面试聊和 Codex Free 额度展示。

##  Bug Fixes / 问题修复

- **Codex Free Weekly Quota**: Codex Free accounts now use the weekly quota window as the effective remaining quota when no 5h window is returned. @mibgb65-cloud
  - Codex Free 账号没有返回 5h 窗口时，现在会使用周额度窗口作为有效剩余额度。
- **Quota Page Display**: The quota page now hides unavailable quota windows, so Free accounts with only weekly limits no longer show an empty 5h quota card. @mibgb65-cloud
  - 额度页现在会隐藏不存在的额度窗口，只有周额度的 Free 账号不再显示空的 5h 额度卡片。

## ⚠️ Breaking Changes / 不兼容更新

- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。

##  Refactor & Performance / 重构与优化

- **Quota Remaining Calculation**: Added a shared effective remaining quota calculation for subscription quota usage, falling back from primary to secondary windows when needed. @mibgb65-cloud
  - 新增订阅额度有效剩余额度计算，在需要时从 primary 窗口回退到 secondary 窗口。
- **Regression Coverage**: Added backend tests for Codex Free secondary-window-only quota parsing and token status recording. @mibgb65-cloud
  - 补充 Codex Free 仅 secondary 窗口额度解析和账号状态记录的后端回归测试。
