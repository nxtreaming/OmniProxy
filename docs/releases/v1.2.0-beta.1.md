# v1.2.0-beta.1 - macOS universal preview and platform support / macOS 通用构建预览与平台适配

This beta starts the `v1.2.0` validation line by extending OmniProxy from a Windows-only desktop release path to an initial macOS implementation. It adds macOS-native platform integrations, DMG-aware update handling, and a GitHub Actions universal DMG build for external testing.

本次 beta 开启 `v1.2.0` 验证版本线，将 OmniProxy 从仅面向 Windows 的桌面发布流程扩展到初步 macOS 实现。版本加入 macOS 原生平台能力、DMG 更新适配和 GitHub Actions universal DMG 构建，用于外部测试。

## Features / 新特性

- **macOS desktop platform support**: Added a Darwin application bundle configuration, macOS menu bar controls, and login launch support through a user LaunchAgent.
  - **macOS 桌面平台支持**：新增 Darwin 应用包配置、macOS 菜单栏控制入口，并通过用户级 LaunchAgent 支持开机自启。
- **macOS credential protection**: macOS credentials are now encrypted using an application master key stored in the current user's Keychain.
  - **macOS 凭据保护**：macOS 上的账号凭据现在使用存放于当前用户 Keychain 中的应用主密钥进行加密。
- **macOS task automation**: Added macOS application opening, browser profile discovery, foreground focus restoration, and keyboard automation support for existing task workflows.
  - **macOS 任务自动化**：为已有任务流程补充 macOS 应用打开、浏览器 Profile 扫描、前台焦点恢复和键盘自动化支持。

## Improvements / 优化

- **DMG-aware in-app updates**: macOS update checks now select `darwin-universal.dmg` assets and matching SHA256 files; after verification, OmniProxy opens the DMG instead of attempting the Windows restart-install flow.
  - **DMG 应用内更新**：macOS 更新检测会选择 `darwin-universal.dmg` 及其 SHA256 文件；校验完成后打开 DMG，而不是走 Windows 的重启安装流程。
- **Platform-specific update UI**: Titlebar and About-page actions now show macOS installation wording, including the drag-to-Applications instruction.
  - **平台化更新界面**：标题栏和关于页更新操作新增 macOS 安装文案，包括拖入 Applications 的提示。
- **Universal build validation**: CI now builds `darwin/universal` on a macOS runner, covering both Apple Silicon and Intel Mac targets in one application bundle.
  - **Universal 构建验证**：CI 现在在 macOS runner 上构建 `darwin/universal`，以一个应用包覆盖 Apple Silicon 与 Intel Mac 目标。

## Release Scope / 发布范围

- **Windows public asset remains available**: The beta tag continues to publish the Windows installer and its SHA256 file through the GitHub Release workflow.
  - **Windows 公开附件保持可用**：该 beta tag 仍通过 GitHub Release 流程发布 Windows 安装包及其 SHA256 文件。
- **macOS unsigned testing DMG**: Beta releases attach an ad-hoc signed universal DMG for external testing. It is not Developer ID signed or notarized, so macOS may show Gatekeeper warnings.
  - **macOS 未签名测试 DMG**：Beta Release 会附加 ad-hoc 签名的 universal DMG 供外部测试。该安装包未使用 Developer ID 签名，也未公证，因此 macOS 可能显示 Gatekeeper 警告。

## Compatibility / 兼容性

- **Attention**: Existing Windows behavior remains unchanged, including DPAPI credential protection and the restart-to-install update handoff.
  - **必读**：现有 Windows 行为保持不变，包括 DPAPI 凭据保护和重启安装更新交接流程。
- **Attention**: The macOS DMG in this beta is for testing only. A signed and notarized public macOS release still requires Apple Developer ID credentials.
  - **必读**：本 beta 中的 macOS DMG 仅用于测试。正式签名并公证的公开 macOS 版本仍需要 Apple Developer ID 凭据。
