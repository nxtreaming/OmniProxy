# v1.1.4-beta.2 - billing preview and drawer polish / 账单预览与抽屉显示优化

This beta continues the `v1.1.4` series with focused desktop UI fixes for billing previews and request-history details. It improves modal motion, keeps the diagnostics drawer inside the content area, and restores the intended light mock-bill preview colors in dark mode.

本次 beta 延续 `v1.1.4` 系列，集中修复费用账单预览和请求历史详情的桌面界面问题。新增预览弹层动效，限制诊断抽屉显示区域，并恢复深色模式下浅色模拟账单模板的正确配色。

## Features / 新特性

- **Billing Preview Motion**: Added the existing modal transition to the simulated billing preview so opening and closing the preview is animated.
  - **账单预览动效**：模拟账单预览接入现有弹层过渡动画，打开和关闭时不再生硬切换。

## Bug Fixes / 问题修复

- **Request Detail Drawer Bounds**: Fixed the request-history detail drawer backdrop so it starts below the title bar and to the right of the sidebar instead of covering the navigation area.
  - **请求详情抽屉边界**：修复请求历史详情抽屉的遮罩位置，使其从标题栏下方和侧边栏右侧开始，不再覆盖导航栏。
- **Mock Bill Preview Colors**: Fixed dark-mode style leakage that made light billing preview templates render with dark gray internal blocks.
  - **模拟账单预览配色**：修复深色模式全局样式污染浅色账单模板的问题，避免预览图内部块变成深灰色。

## Breaking Changes / 不兼容更新

- **Attention**: No backend API or local data format breaking changes are expected in this beta release.
  - **必读**：本 beta 版本预计没有后端 API 或本地数据格式不兼容更新。
