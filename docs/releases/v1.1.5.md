# v1.1.5 - Gemini-style desktop refresh / Gemini 风格桌面界面重构

This stable release focuses on the OmniProxy desktop experience. It unifies the application around a cleaner Gemini-style visual system, improves page-level scrolling behavior, and polishes high-density operational views such as account quota, request history, realtime logs, billing, settings, and OpenRouter chat.

本正式版聚焦 OmniProxy 桌面端体验，将应用统一到更整洁的 Gemini 风格视觉体系，优化子页面滚动行为，并重点打磨账号额度、请求历史、实时日志、费用账单、全局设置和 OpenRouter 对话等高密度操作页面。

## Features / 新特性

- **Gemini-style UI system**: Refreshed the desktop shell, sidebar, cards, buttons, dialogs, dropdowns, scrollbars, and global snackbars with a consistent minimal light/dark style.
  - **Gemini 风格 UI 体系**：统一桌面外壳、侧边栏、卡片、按钮、弹窗、下拉框、滚动条和全局消息提示，形成一致的极简浅色 / 深色主题。
- **Per-page scroll memory**: Each workspace page now keeps its own scroll position instead of sharing one global scroll offset.
  - **子页面滚动记忆**：每个工作区页面独立记录滚动位置，不再共用同一个全局滚动状态。
- **Realtime log stream**: The realtime log page now focuses on the latest 5 minutes of events and scrolls inside the event panel, keeping the overall page stable.
  - **实时日志流**：实时日志页面只聚焦最近 5 分钟事件，并在事件面板内部滚动，避免整页被日志列表拉长。
- **Responsive desktop bounds**: Added a wider minimum desktop width to prevent dense dashboard panels from collapsing into unreadable narrow columns.
  - **桌面最小宽度约束**：提高桌面窗口最小宽度，避免仪表盘等高密度页面在窄窗口下被压缩成不可读的竖列。

## Improvements / 优化

- **Dashboard and usage trends**: Rebalanced dashboard stat cards, recent logs, activity heatmap, daily usage summaries, and usage trend panels for consistent card heights and spacing.
  - **仪表盘与用量趋势**：重新平衡状态卡片、最近日志、请求活跃日历、分天用量统计和趋势面板的高度与间距。
- **Account and quota views**: Simplified provider filters, account cards, quota details, progress bars, action buttons, and OpenRouter quota summaries to reduce visual clutter.
  - **账号与额度页面**：优化厂商筛选、账号卡片、额度明细、进度条、操作按钮和 OpenRouter 额度展示，减少视觉噪音。
- **Request history and diagnostics**: Updated filter controls, table scrolling, horizontal overflow handling, and request-detail drawer styling for better readability.
  - **请求历史与诊断详情**：优化筛选控件、表格内部滚动、横向溢出处理和请求详情抽屉样式，提高可读性。
- **Billing and settings pages**: Reduced nested card layers, improved mock-bill previews, model cost details, one-click setup sections, and global settings layout.
  - **费用账单与设置页面**：减少卡片嵌套层级，优化模拟账单预览、模型费用明细、一键配置分区和全局设置布局。
- **OpenRouter chat**: Refined the chat test layout so model selection, quota status, conversation area, and input controls occupy a more compact desktop workspace.
  - **OpenRouter 对话**：压缩模型选择、额度状态、对话区域和输入控件的布局占用，让桌面试聊页面更紧凑。

## Bug Fixes / 问题修复

- **Scrollbar jitter**: Fixed page-width shifts caused by appearing scrollbars and removed unnecessary sidebar scrolling.
  - **滚动条抖动**：修复滚动条出现时导致页面闪动的问题，并移除侧边栏不必要的滚动条。
- **Realtime log item spacing**: Fixed metadata chips such as status, latency, account, and time so they stay grouped with the corresponding log message.
  - **实时日志行距**：修复状态码、耗时、账号和时间等信息没有跟随日志消息整体包裹的问题。
- **Dialog backgrounds**: Normalized modal and drawer backgrounds, including API key import, token editor, billing preview, and request diagnostics.
  - **弹窗背景**：统一 API Key 批量导入、账号编辑、账单预览和请求诊断等弹窗 / 抽屉背景。
- **Light theme contrast**: Fixed low-contrast light-theme buttons and disabled icons in setup and update-related actions.
  - **浅色模式对比度**：修复一键配置和更新相关按钮在浅色模式下文字或图标过淡的问题。

## Compatibility / 兼容性

- **Attention**: No backend API, proxy route, credential format, or local data format breaking changes are expected in this stable release.
  - **必读**：本正式版预计没有后端 API、代理路由、凭据格式或本地数据格式不兼容更新。
