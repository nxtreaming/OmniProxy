# v1.2.5-beta.4 - Reset-credit details placement / 刷新卡详情定位修复

This beta fixes Codex quota reset-credit history appearing at the bottom of the quota page instead of inside its account card.

本次 beta 修复 Codex 额度刷新卡记录错误显示在额度页底部、而不是所属账号卡片内的问题。

## Features / 新特性

- None / 无

## Bug Fixes / 问题修复

- **Account-local reset-credit history**: Clicking **View records** now expands the history inside the corresponding Codex account card. Clicking it again collapses the same card.
  - **账号内展开刷新卡记录**：点击「查看记录」后，记录会在对应 Codex 账号卡片内展开；再次点击即收起。
- **Removed misplaced page-level content**: Removed the unsupported drawer rendering path that exposed detail content as a full-width block below all accounts.
  - **移除错位的页面级内容**：移除未正常生效的抽屉渲染路径，不再将详情作为全宽区块显示在所有账号下方。
- **Accessible expansion state**: The record control now exposes synchronized expanded state, visible focus, responsive layout, and reduced-motion behavior.
  - **可访问的展开状态**：记录按钮现会同步展开状态，并保留焦点、响应式布局和减少动画适配。
