# v1.0.9 - Billing, quota controls, and expanded provider support / 账单、额度控制与更多厂商支持

This release adds local billing reports, per-account enable switches, GLM API and Coding Plan quota handling, Claude OAuth import, and improved one-click client configuration.
本次版本新增本地费用账单、账号启停控制、GLM API 与 Coding Plan 额度适配、Claude OAuth 导入，以及更完整的一键配置能力。

## Features / 新特性
- **Billing Reports**: Add a billing page with selectable dates, local daily usage retention, configurable history retention, manual cleanup, and invoice-style PNG export.
  - 新增「费用账单」页面，支持选择历史日期、保存每日 Token 用量、设置保留天数、手动清理，并导出发票风格 PNG。
- **Pricing Estimates**: Add default model price estimates for common OpenAI, Claude, DeepSeek, Gemini, Kimi, GLM, MiniMax, and MiMo models. Only priced model calls are counted in billing totals.
  - 增加常见模型默认价格估算；账单只统计能匹配价格的模型调用，健康检查和额度刷新不会计入费用。
- **Account Enable Switches**: Add per-account enable/disable controls in the quota page. Disabled accounts are skipped by routing, balanced scheduling, health checks, and automatic quota refresh.
  - 额度页支持单独启用或停用账号；停用账号不会参与代理调度、均衡选择、健康检查和自动额度刷新。
- **GLM Credentials**: Split Zhipu GLM accounts into API Key and Coding Plan credential types, with separate quota logic and dashboard status.
  - 智谱 GLM 账号区分 API Key 与 Coding Plan，分别适配额度查询逻辑，并在仪表盘/额度页展示状态。
- **GLM Balance Packages**: Show API Key resource package details including package name, remaining amount, type, expiration time, and applicable model. Only effective Token packages count toward Token balance; TIMES packages are displayed but not counted.
  - GLM API Key 额度展示资源包明细，包括名称、剩余额度、类型、到期时间和适用模型；只有有效 Token 包计入 Token 余额，次数包仅展示。
- **Claude OAuth Import**: Support importing Claude OAuth JSON credentials for Anthropic accounts.
  - Anthropic 账号支持导入 Claude OAuth JSON 凭据。
- **Client Quickstart**: Expand one-click configuration for Claude Code, Codex, Gemini CLI, OpenCode, Xiaomi MiMo, DeepSeek, Kimi, and Zhipu GLM. Zhipu Claude Code configuration uses `glm-5.1`.
  - 扩展一键配置能力，覆盖 Claude Code、Codex、Gemini CLI、OpenCode、MiMo、DeepSeek、Kimi 与 GLM；GLM Claude Code 默认使用 `glm-5.1`。

## Bug Fixes / 问题修复
- **Billing Date Selection**: Allow selecting earlier dates instead of only the current day.
  - 费用账单支持选择历史日期，不再只能查看当天。
- **Billing Layout**: Fix overflow and alignment issues in the invoice-style billing view and model fee table.
  - 修复账单估算卡片和模型费用明细的溢出、对齐问题。
- **Quota Display**: Show weekly limit reached state for accounts whose plan window has been exhausted.
  - 当账号达到周限额时，在仪表盘/额度页明确显示限额状态。
- **Enable Control**: Make quota-page account enable/disable controls reliably clickable with both a switch and explicit text button.
  - 修复额度页账号启用/停用控件不易点击或不触发的问题。
- **Usage Accounting**: Exclude health checks, quota refreshes, and token validation requests from billing usage.
  - 账单用量排除健康检查、额度刷新和 Token 验证请求。
