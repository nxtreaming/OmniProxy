# v1.2.3-beta.4 - Dashboard and quota UI polish / 仪表盘与额度页界面优化

This beta follows `v1.2.3-beta.3` with focused UI fixes for the dashboard active-request layout and Codex workspace quota cards.

本次 beta 在 `v1.2.3-beta.3` 基础上，集中修复仪表盘活跃请求布局，并优化 Codex 多工作区额度卡片的展示质感。

## Features / 新特性

- None / 无

## Bug Fixes / 问题修复

- **Dashboard layout**: Stabilized the dashboard grid so the tool-account usage panel and active-request panel stay aligned when live requests are present.
  - **仪表盘布局**：稳定仪表盘网格布局，存在活跃请求时“编程工具账号占用”和“活跃请求”面板会保持对齐，不再上下错位并留下大面积空白。
- **Dashboard responsive rules**: Let the dashboard component own its 12-column layout and removed the generic full-width class from the tool usage panel to avoid global CSS conflicts.
  - **仪表盘响应式规则**：由仪表盘组件自身接管 12 列布局，并移除工具占用面板的通用全宽类，避免被全局 CSS 覆盖。

## Refactor & Performance / 重构与优化

- **Quota workspace switcher**: Refined the same-email Codex workspace switcher into a compact status-row capsule, with quieter borders, smaller controls, and less visual interruption inside quota cards.
  - **额度工作区切换**：将同邮箱 Codex 工作区切换器优化为状态行内的紧凑胶囊控件，边框更克制、按钮更小，减少对额度卡片主信息的干扰。
