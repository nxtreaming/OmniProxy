# v1.1.7-beta.1 - quota card spacing polish / 额度卡片间距优化

This beta starts the `v1.1.7` line with a focused quota-page layout polish. It gives account quota cards more horizontal room on desktop widths so three-column layouts no longer make subscription and usage panels feel cramped.

本次 beta 开启 `v1.1.7` 版本线，集中优化额度页布局。在桌面宽度下为账号额度卡片保留更多横向空间，避免三列布局让订阅额度和代理用量面板显得过于拥挤。

## Improvements / 优化

- **Roomier quota cards**: The quota page now prefers wider account cards, so the screenshot-width desktop layout falls back to two columns instead of squeezing three cards into one row.
  - **额度卡片更宽松**：额度页优先使用更宽的账号卡片，截图这类桌面宽度会回落到两列，而不是一行挤三张卡片。
- **Ultra-wide behavior preserved**: Very wide desktop windows can still use a multi-column grid, but only after each card has enough room for its internal quota panels.
  - **保留超宽屏多列**：超宽窗口仍可使用多列网格，但前提是每张卡片内部额度面板拥有足够空间。

## Bug Fixes / 问题修复

- **Cramped quota page**: Fixed the quota account grid using three narrow cards at common desktop widths, which made action buttons, quota windows, and proxy-usage stats feel crowded.
  - **额度页过挤**：修复常见桌面宽度下一行显示三张窄卡片的问题，避免操作按钮、额度窗口和代理用量统计被压得太紧。

## Compatibility / 兼容性

- **Attention**: No proxy route, credential format, local data format, or Wails binding changes are expected in this beta release.
  - **必读**：本 beta 预计没有代理路由、凭据格式、本地数据格式或 Wails 绑定不兼容更新。
