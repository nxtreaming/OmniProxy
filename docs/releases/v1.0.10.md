# v1.0.10 - Desktop UI and mock billing / 桌面界面与模拟账单

This stable release consolidates the `v1.0.10` beta series into a production build. It refreshes the desktop shell, improves account management dialogs, adds richer mock billing image templates, and keeps the installer on the normal production profile.
本正式版整合 `v1.0.10` beta 系列改动，包含桌面外壳刷新、账号管理弹窗优化、更丰富的模拟账单图片模板，并保持安装包使用正常生产 profile。

## Changes / 更新内容
- **Desktop Shell**: Refresh the desktop layout with denser navigation, a custom draggable title bar, and clearer window controls.
  - **桌面外壳**: 刷新桌面布局，加入更紧凑的导航、自定义可拖动标题栏和更清晰的窗口控制。
- **Message Readability**: Make toast, notification, and confirmation dialogs use solid backgrounds, stronger borders, and deeper shadows.
  - **提示可读性**: 提示、通知和确认框改为实色背景、更清晰边框和更明显阴影，避免背景内容干扰阅读。
- **Account Management**: Replace the browser-native delete confirmation with an in-app destructive confirmation dialog and add account modal enter/exit motion.
  - **账号管理**: 删除账号确认改为应用内危险操作弹窗，并为添加/编辑账号弹窗增加进入和退出动画。
- **Mock Billing**: Add multiple mock bill image templates including poster, neon, receipt, clean card, meter, and longform data layouts.
  - **模拟账单**: 新增费用海报、午夜霓虹、复古小票、清爽卡片、能量仪表和数据长图等多种模拟账单模板。
- **Bill Preview and Language**: Add preview-before-save for generated PNG bills and support Chinese/English bill content output.
  - **账单预览与语言**: 生成 PNG 前支持预览，账单图片内容支持中文和英文输出。
- **Layout Polish**: Reorganize the billing header and template controls for clearer scanning and fewer awkward jumps.
  - **布局细节**: 调整账单顶部和模板控制区，提升扫视效率并减少生硬跳动。

## Bug Fixes / 问题修复
- **Billing Preview Stability**: Avoid page blanking after bill preview/export by keeping preview state inside the billing view and using a safer save flow.
  - **账单预览稳定性**: 调整预览和保存流程，避免预览或导出账单图片后其它页面内容变空。
- **Release Packaging**: Ensure beta and stable installers use the production installer profile unless explicitly building a local dev executable.
  - **发布打包**: 确保 beta 和正式安装包使用生产安装包 profile，只有本地开发 exe 才使用 dev profile。

## Breaking Changes / 不兼容更新
- **Attention**: No backend API or data format breaking changes are expected in this release.
  - **必读**: 本版本预计没有后端接口或数据格式不兼容更新。
