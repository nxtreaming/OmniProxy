# v1.0.10-beta.1 - Desktop UI beta / 桌面界面 beta

This beta release focuses on the OmniProxy desktop interface. It keeps the existing proxy, account, quota, billing, and logging behavior unchanged while refreshing the app shell and dashboard layout for validation before a future stable release.
本次 beta 版本聚焦 OmniProxy 桌面界面，不改变现有代理、账号、额度、账单和日志行为，主要用于验证后续正式版前的界面调整。

## Beta Notes / Beta 说明
- **Prerelease**: This build is intended for UI validation before the stable `v1.0.10` release.
  - **预发布**: 此版本用于正式 `v1.0.10` 前的界面验证。
- **Coexistence**: Development builds still use the dev profile defaults, including the `.omniproxy-dev` data directory and `3001/3891` ports.
  - **共存**: 开发构建仍使用 dev profile 默认配置，包括 `.omniproxy-dev` 数据目录和 `3001/3891` 端口。

## Changes / 更新内容
- **Desktop Shell**: Group the left navigation into overview, runtime, and system sections.
  - **桌面外壳**: 左侧导航按「总览」「运行」「系统」分组。
- **Dashboard Layout**: Replace the previous dashboard composition with a compact status strip, denser metrics, and a 12-column content grid.
  - **仪表盘布局**: 使用更紧凑的状态条、指标卡和 12 栅格内容区重组首页。
- **Visual Tuning**: Reduce decorative gradients and oversized hero treatments in favor of a quieter native utility style.
  - **视觉调整**: 降低装饰性渐变和大横幅感，转向更安静的桌面工具风格。
- **Interaction Polish**: Refine hover, focus, table, progress, card, and sidebar states without changing backend contracts.
  - **交互细节**: 调整 hover、focus、表格、进度条、卡片和侧栏状态，不改变后端接口。

## Breaking Changes / 不兼容更新
- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。
