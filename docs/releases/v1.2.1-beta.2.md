# v1.2.1-beta.2 - Codex weekly quota estimate / Codex 周额度美元预估

This beta continues the `v1.2.1` validation line with a focused Codex quota display update. It adds an estimated weekly USD quota value based on observed token usage and the current weekly remaining percentage, and corrects Codex Team accounts to show the primary window as `5h额度`.

本次 beta 继续验证 `v1.2.1` 版本线，聚焦 Codex 额度展示。版本新增基于已使用 Token 与当前周额度剩余百分比推测的周额度美元预估，并修复 Codex Team 账号主窗口应显示为 `5h额度` 的问题。

##  Features / 新特性

- **Codex quota estimate**: Added an estimated weekly USD quota display for Codex subscription accounts, calculated from the account's recorded token usage and weekly remaining percentage.
  - **Codex 额度预估**：为 Codex 订阅账号新增周额度美元预估展示，根据账号已记录 Token 用量和周额度剩余百分比推算。

##  Bug Fixes / 问题修复

- **Codex Team quota label**: Fixed Codex Team accounts to show the primary quota window as `5h额度` instead of `本月额度`.
  - **Codex Team 额度标签**：修复 Codex Team 账号主额度窗口误显示为 `本月额度` 的问题，现在显示为 `5h额度`。

## ⚠️ Breaking Changes / 不兼容更新

- None / 无

##  Refactor & Performance / 重构与优化

- **Frontend regression coverage**: Added frontend utility tests for the Codex Team quota label and weekly USD estimate behavior.
  - **前端回归覆盖**：新增 Codex Team 额度标签和周额度美元预估的前端工具函数测试。
