# v1.2.3-beta.3 - Quota workspace grouping / 额度页工作区聚合

This beta improves the quota page for Codex accounts that share one ChatGPT email across multiple `account_id` workspaces.

本次 beta 优化同一个 ChatGPT 邮箱下存在多个 `account_id` 工作区时的额度页展示。

## Features / 新特性

- **Quota workspace grouping**: Codex credentials with the same email are now shown as one account card on the quota page, with left and right controls to switch between workspace `account_id` values.
  - **额度工作区聚合**：额度页现在会把同邮箱 Codex 凭证显示为一个账号卡片，并提供左右按钮切换不同工作区的 `account_id`。
- **Workspace-aware actions**: Selecting an account or refreshing quota still targets the currently displayed workspace credential, keeping routing and quota refresh behavior precise.
  - **工作区感知操作**：选择账号或刷新额度仍然作用于当前展示的工作区凭证，保证路由选择和额度刷新不会串账号。

## Refactor & Performance / 重构与优化

- **Quota display model**: Added a small tested grouping helper for quota account cards so API keys and different emails remain separate while same-email Codex workspaces can be folded together.
  - **额度展示模型**：新增带测试的额度卡片分组工具，API Key 和不同邮箱保持独立，同邮箱 Codex 工作区可折叠展示。
