# v1.1.3-beta.2 - API balance totals and dashboard log fix / API 余额汇总与仪表盘日志修复

This beta improves API-key quota visibility and fixes a compact dashboard log layout issue in the desktop UI.
本次 beta 优化 API Key 额度可见性，并修复桌面端仪表盘紧凑日志布局问题。

## Features / 新特性

- **API Balance Totals**: Provider quota and account pages now show total API Key balances grouped by currency, so CNY and USD balances are counted separately.
  - **API 总额度**：厂商额度页和账号管理页按币种汇总 API Key 余额，人民币和美元分开统计。
- **DeepSeek Multi-Currency Balance Details**: DeepSeek API Key balance refresh now preserves all returned currency balances for separate aggregation.
  - **DeepSeek 多币种余额明细**：DeepSeek API Key 刷新余额时保留上游返回的所有币种，用于分别汇总。

## Fixes / 修复

- **Dashboard Recent Logs**: Fixed compact recent-log rows inheriting flex layout, which could squeeze model/client labels, message text, and timestamps together.
  - **仪表盘最近日志**：修复紧凑日志行继承 flex 布局导致标签、正文和时间挤在一起的问题。
