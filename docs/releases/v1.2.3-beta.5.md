# v1.2.3-beta.5 - Quota workspace controls / 额度工作区控件优化

This beta follows `v1.2.3-beta.4` with final quota-page control polish and a dashboard layout simplification.

本次 beta 在 `v1.2.3-beta.4` 基础上，继续优化额度页工作区控件，并简化仪表盘布局。

## Features / 新特性

- **Workspace select all**: Same-email Codex workspace groups on the quota page now provide a `全选` action to select all enabled workspaces under that account.
  - **工作区全选**：额度页同邮箱 Codex 工作区分组新增 `全选` 操作，可一次选中该账号下所有已启用工作区。

## Bug Fixes / 问题修复

- **Quota workspace row**: Aligned workspace number dots and the workspace switcher on the same row so multi-workspace cards no longer look split across two control bands.
  - **额度工作区行**：将工作区数字圆点和工作区切换器对齐到同一行，避免多工作区卡片控件上下割裂。
- **Dashboard layout**: Removed the dashboard active-request panel that could disturb the grid when live requests were present; the current connection metric remains available in the top cards.
  - **仪表盘布局**：移除会在有实时请求时扰乱网格的“活跃请求”面板，顶部“当前连接”统计仍保留。

## Refactor & Performance / 重构与优化

- **Dashboard cleanup**: Removed the unused active-request panel helpers and styles after the layout simplification.
  - **仪表盘清理**：布局简化后同步移除未使用的活跃请求面板辅助逻辑和样式。

- None / 无
