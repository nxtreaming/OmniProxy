# v1.0.10-beta.3 - Window chrome beta / 窗口体验 beta

This beta release focuses on desktop window polish and message readability. It introduces a custom draggable title bar for the Wails shell and makes toast/message dialogs visually solid enough to avoid text bleeding through from the page behind them.
本次 beta 版本聚焦桌面窗口体验和提示信息可读性，加入 Wails 自定义可拖动标题栏，并让提示框与确认框使用更实的背景，避免透出后方页面文字。

## Beta Notes / Beta 说明
- **Prerelease**: This build is intended for desktop shell validation before the stable `v1.0.10` release.
  - **预发布**: 此版本用于正式 `v1.0.10` 前验证桌面窗口外壳体验。
- **Coexistence**: Prerelease installers are now built with the dev profile tag, keeping the Dev data directory and ports separate from the production version.
  - **共存**: 预发布安装包现在使用 dev profile 构建标签，继续与正式版的数据目录和端口隔离。

## Changes / 更新内容
- **Custom Title Bar**: Replace the native window chrome with a custom top bar that supports dragging, double-click maximise, minimise, maximise/restore, and close-to-tray behavior.
  - **自定义标题栏**: 使用自定义顶部操作栏替代原生窗口栏，支持拖动、双击最大化、最小化、最大化/还原以及关闭到托盘。
- **Window Styling**: Enable frameless Wails mode so the desktop shell can own the top window controls.
  - **窗口样式**: 启用 Wails 无边框窗口，让应用自身控制顶部窗口操作区域。
- **Message Readability**: Make toast, message, notification, and confirmation dialogs use solid backgrounds, stronger borders, and deeper shadows.
  - **提示可读性**: 提示、通知和确认框改为实色背景、更清晰边框和更明显阴影，避免背景文字干扰阅读。
- **Release Packaging**: Build prerelease tags with `omniproxy_dev` so beta installers behave like Dev builds.
  - **发布打包**: 预发布 tag 自动使用 `omniproxy_dev` 构建，确保 beta 安装包按 Dev 版本运行。

## Breaking Changes / 不兼容更新
- **Attention**: No backend API or data format breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有后端接口或数据格式不兼容更新。
