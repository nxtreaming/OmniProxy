# v1.1.8-beta.2 - Dashboard usage and settings fixes / 仪表盘用量与设置修复

This beta continues the `v1.1.8` validation line with fixes for the Windows desktop app. It corrects the Beta update setting interaction, improves the account delete confirmation backdrop, and changes dashboard proxy token totals to use an account-independent billing summary.

本次 beta 继续验证 `v1.1.8` 版本线，集中修复 Windows 桌面端问题：修复 Beta 更新设置开关不可操作、优化账号删除确认弹窗背景，并将仪表盘代理 Token 累计改为独立账单汇总，不再随账号删除而减少。

## Fixes / 修复

- **Dashboard proxy token totals**: The dashboard now reads total proxy requests and tokens from an independent billing summary instead of summing only currently configured accounts.
  - **仪表盘代理 Token 累计**：仪表盘改为读取独立账单汇总，不再只汇总当前账号列表。
- **Deleted accounts no longer remove totals**: Clearing request history or deleting accounts no longer reduces the lifetime proxy token total; only clearing billing summaries resets that statistic.
  - **删除账号不再影响累计用量**：清空请求历史或删除账号不会减少累计代理 Token；只有清空账单汇总会重置该统计。
- **Beta update switch interaction**: The Beta update setting row now has a reliable clickable area and an accessible switch label.
  - **Beta 更新开关交互**：Beta 更新设置项的点击区域已修复，并补充开关可访问标签。
- **Account delete dialog backdrop**: The account deletion confirmation dialog and dark-mode backdrop now use clearer surfaces and avoid the overly dark footer strip.
  - **账号删除弹窗背景**：账号删除确认弹窗和深色背景层调整为更清晰的表面色，避免底部区域过暗。

## Compatibility / 兼容性

- **Attention**: Existing daily billing summaries are used to initialize the new lifetime billing summary when the app starts after upgrading.
  - **必读**：升级后会使用已有每日账单汇总初始化新的累计账单统计。
- **Attention**: Historical usage that was already deleted or pruned before this version cannot be reconstructed automatically.
  - **必读**：升级前已经被清空或裁剪掉的历史用量无法自动恢复。
- **Attention**: No proxy route or credential format breaking changes are expected in this beta release.
  - **必读**：本 beta 版本预计没有代理路由或凭据格式不兼容更新。
