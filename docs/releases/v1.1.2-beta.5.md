# v1.1.2-beta.5 - usage trends and release notes layout / 用量趋势与版本说明布局

This beta adds a dedicated usage trends page for all recorded daily usage, showing both token volume and request counts as line charts. It also expands the About page release notes panel to use the full page width so longer bilingual notes are easier to read.
本次 beta 新增独立的用量趋势页面，用折线图展示全部已记录天数的 Token 用量和请求次数；同时将关于应用里的版本说明扩展为满宽面板，让较长的双语发布说明更易阅读。

## Features / 新特性

- **Usage Trends Page**: Added a new “用量趋势” page under the overview navigation, backed by the existing daily proxy usage records.
  - **用量趋势页面**：在总览导航中新增“用量趋势”页面，复用现有每日代理用量数据。
- **Full Daily Token Chart**: The trend page renders all recorded daily token totals as a line and area chart instead of only showing the dashboard's recent 7-day summary.
  - **完整 Token 趋势**：趋势页以折线和面积图展示全部已记录天数的 Token 总量，不再局限于仪表盘最近 7 天摘要。
- **Request Count Trend**: The same page also shows request counts as a separate line chart, plus totals, recent 7-day comparison text, peak day, and a daily detail table.
  - **请求次数趋势**：同页新增请求次数折线图，并展示总量、近 7 天对比、峰值日期和每日明细表。

## UX Polish / 体验优化

- **Dashboard Entry Point**: The dashboard daily usage panel now links to the full usage trends page while keeping billing details available as a separate action.
  - **仪表盘入口**：仪表盘分天用量区域新增完整趋势入口，同时保留账单明细入口。
- **Release Notes Full Width**: The About page release notes now occupy the full page width below the version/runtime cards, avoiding empty right-side space.
  - **版本说明满宽**：关于应用页的版本说明移动到版本与运行信息卡片下方，并占据整页宽度，避免右侧留白。
- **Wide Reading Layout**: Long release notes use a two-column layout on wide screens and automatically collapse to one column on narrower layouts.
  - **宽屏阅读布局**：较长版本说明在宽屏下按两列排版，窄屏下自动收敛为单列。

## Breaking Changes / 不兼容更新

- **Attention**: No backend API or local data format breaking changes are expected in this beta release.
  - **必读**：本 beta 版本预计没有后端 API 或本地数据格式不兼容更新。
