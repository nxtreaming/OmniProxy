# v1.0.12-beta.2 - MiMo Claude context beta / MiMo Claude 上下文 beta

This beta release focuses on the Xiaomi MiMo Claude Code quickstart. It enables the 1M-context MiMo V2.5 Pro model as the default Claude Code Opus route while keeping the regular MiMo V2.5 Pro model available as an explicit custom option.
本次 beta 版本聚焦 Xiaomi MiMo 的 Claude Code 一键配置，将 1M 上下文的 MiMo V2.5 Pro 作为 Claude Code Opus 默认路由，同时保留普通 MiMo V2.5 Pro 作为可显式选择的自定义模型。

## Beta Notes / Beta 说明

- **Prerelease**: This build is intended for validating the MiMo Claude Code model configuration before a future stable release.
  - **预发布**: 此版本用于在后续正式版前验证 MiMo Claude Code 模型配置。
- **Model Choice**: The default MiMo Pro route now uses `mimo-v2.5-pro[1m]`; `mimo-v2.5-pro` remains available for users who do not want the 1M-context route.
  - **模型选择**: MiMo Pro 默认路由改为 `mimo-v2.5-pro[1m]`；不需要 1M 上下文的用户仍可使用 `mimo-v2.5-pro`。

## Improvements / 优化

- **MiMo Claude Quickstart**: Claude Code one-click MiMo configuration now writes `ANTHROPIC_MODEL` and `ANTHROPIC_DEFAULT_OPUS_MODEL` to `mimo-v2.5-pro[1m]`.
  - **MiMo Claude 一键配置**: Claude Code 的 MiMo 一键配置现在会将 `ANTHROPIC_MODEL` 和 `ANTHROPIC_DEFAULT_OPUS_MODEL` 写为 `mimo-v2.5-pro[1m]`。
- **Regular Pro Option**: The same configuration keeps `mimo-v2.5-pro` as `ANTHROPIC_CUSTOM_MODEL_OPTION` so the non-1M Pro model is still available.
  - **普通 Pro 选项**: 同一配置会将 `mimo-v2.5-pro` 保留为 `ANTHROPIC_CUSTOM_MODEL_OPTION`，普通 Pro 模型仍可使用。
- **Router Compatibility**: The Anthropic router tests now cover both `mimo-v2.5-pro[1m]` and `mimo-v2.5-pro` to ensure both model IDs route to Xiaomi MiMo unchanged.
  - **路由兼容**: Anthropic 路由测试同时覆盖 `mimo-v2.5-pro[1m]` 与 `mimo-v2.5-pro`，确保两个模型 ID 都会原样路由到 Xiaomi MiMo。

## Bug Fixes / 问题修复

- **MiMo Pro Context**: Fixed the Claude Code MiMo preset not enabling the `[1m]` context variant for the Pro model.
  - **MiMo Pro 上下文**: 修复 Claude Code MiMo 预设未为 Pro 模型启用 `[1m]` 上下文变体的问题。

## Breaking Changes / 不兼容更新

- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。
