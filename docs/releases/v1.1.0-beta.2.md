# v1.1.0-beta.2 - TokenRouter beta / TokenRouter 接入 beta

This beta release adds TokenRouter as a first-class OmniProxy provider and extends the Pi/OpenCode local client configuration to use TokenRouter auto-routing models.
本次 beta 版本新增 TokenRouter 一等 provider，并扩展 Pi/OpenCode 本地客户端配置，可直接使用 TokenRouter 自动路由模型。

##  Features / 新特性

- **TokenRouter Provider**: Added TokenRouter API Key account management, `tr_` key validation, configurable Base URL, and local OpenAI-compatible route `/tokenrouter/v1`. @mibgb65-cloud
  - 新增 TokenRouter API Key 账号管理、`tr_` 前缀校验、可配置 Base URL，以及本地 OpenAI 兼容入口 `/tokenrouter/v1`。
- **Pi/OpenCode Routing**: Added TokenRouter auto models such as `auto:balance`, `auto:quality`, `auto:speed`, and `auto:cost` to local client configuration. @mibgb65-cloud
  - 为 Pi/OpenCode 本地配置新增 TokenRouter auto 模型，例如 `auto:balance`、`auto:quality`、`auto:speed` 和 `auto:cost`。
- **TokenRouter Validation**: Added TokenRouter credential validation through routing rules and compatibility for Base URLs with or without `/v1`. @mibgb65-cloud
  - 新增通过 routing rules 校验 TokenRouter 凭据，并兼容 Base URL 是否包含 `/v1`。

##  Bug Fixes / 问题修复

- **Client Logs**: TokenRouter direct requests are now labeled as TokenRouter in local request logs instead of generic API clients. @mibgb65-cloud
  - TokenRouter 直连请求现在会在本地请求日志中标记为 TokenRouter，而不是泛化为 API Client。

## ⚠️ Breaking Changes / 不兼容更新

- **Attention**: No breaking changes are expected in this beta release.
  - **必读**: 本 beta 版本预计没有不兼容更新。

##  Refactor & Performance / 重构与优化

- **Provider Configuration**: Updated frontend settings, Wails bindings, README docs, and route tests for the new TokenRouter provider. @mibgb65-cloud
  - 更新前端设置、Wails 绑定、README 文档和路由测试，以支持新的 TokenRouter provider。
