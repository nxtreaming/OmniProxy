# v1.1.3-beta.1 - desktop client integrations and setup polish / 桌面客户端接入与一键配置优化

This beta starts the `v1.1.3` series with new local client integrations and a cleaner Claude setup workflow. It adds DeepSeek-TUI configuration, Claude Desktop gateway profile support, separates Claude Code CLI and Desktop usage tracking, and removes the unstable Claude Desktop language injection experiment.
本次 beta 开始 `v1.1.3` 系列，重点补充本地客户端接入与 Claude 配置体验。新增 DeepSeek-TUI 一键配置、Claude Desktop Gateway Profile 接入，区分 Claude Code CLI 与 Desktop 统计，并移除不稳定的 Claude Desktop 语言注入实验。

## Features / 新特性

- **DeepSeek-TUI Integration**: Added one-click configuration for DeepSeek-TUI, writing its local config to route requests through OmniProxy.
  - **DeepSeek-TUI 接入**：新增 DeepSeek-TUI 一键配置，将本地配置指向 OmniProxy。
- **Claude Desktop Gateway**: Added Claude Desktop model routing through a local 3P Gateway Profile, allowing users to choose the models exposed to the Desktop app.
  - **Claude Desktop Gateway**：新增 Claude Desktop 3P Gateway Profile 接入，可按选择写入 Desktop 可用模型。
- **Separate Claude Tool Tracking**: Claude Code CLI and Claude Desktop now use separate client/tool attribution so request usage is easier to tell apart.
  - **Claude 工具分开统计**：Claude Code CLI 与 Claude Desktop 使用独立工具标识，便于区分请求来源。

## UX Polish / 体验优化

- **Quickstart Order**: The one-click setup page follows the requested order: Codex, Claude Code, Gemini, OpenCode, Pi, DeepSeek-TUI.
  - **一键配置顺序**：按 Codex、Claude Code、Gemini、OpenCode、Pi、DeepSeek-TUI 排列。
- **Claude Setup Layout**: The Claude setup area now separates selected-model writes from quick single-model presets, with cleaner model cards and grouped actions.
  - **Claude 配置布局**：将“按当前选择写入”和“快捷单模型”分组展示，模型选择改为更清晰的卡片式布局。
- **Removed Unstable Language Button**: Removed the experimental Claude Desktop Chinese language button because WindowsApps resource patching is not reliable across installs.
  - **移除不稳定语言按钮**：移除实验性的 Claude Desktop 中文按钮，避免在不同 WindowsApps 安装形态下产生无效操作。

## Breaking Changes / 不兼容更新

- **Attention**: No backend API or local data format breaking changes are expected in this beta release.
  - **必读**：本 beta 版本预计没有后端 API 或本地数据格式不兼容更新。
